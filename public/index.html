FAQ

Where does data live?

Editing uses browser storage. Clicking Share sends JSON to PHP and saves it in SQLite/MySQL.

How does /s/{id} work?

Apache rewrite routes /s/{id} to a PHP page which injects JSON for the frontend viewer.

Is it mobile-friendly?

Yes. Layout stacks on small screens; schedule grid scrolls horizontally.

Can we add accounts later?

Yes. Keep MVP: share links first, accounts later.
